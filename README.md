# disordered-hilbert

> Removing the need for choosing a total ordering on the variables in the
> Hilbert’s proof of Nœtherianity for rings of polynomials.


## Build

```bash
make disordered.pdf
```

## Dependencies

- `make`
- `pdflatex`
- `bibtex`
- `latexmk`
- `pandoc`
